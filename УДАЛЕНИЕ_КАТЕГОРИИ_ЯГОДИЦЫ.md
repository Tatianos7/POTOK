# Удаление категории "Ягодицы" из базы данных

## Что нужно сделать:

Если категория "Ягодицы" все еще отображается в приложении, выполните одно из действий:

### Способ 1: Автоматическое удаление (рекомендуется)

1. Откройте страницу импорта упражнений: `/import-exercises`
2. Seed скрипт автоматически:
   - Найдет категорию "Ягодицы" в базе данных
   - Удалит все упражнения из этой категории
   - Удалит связи упражнений с мышцами
   - Удалит саму категорию "Ягодицы"
3. После этого категория исчезнет из списка

### Способ 2: Вручную через Supabase SQL Editor

Если автоматическое удаление не сработало, выполните SQL скрипт в Supabase SQL Editor:

1. Откройте Supabase Dashboard
2. Перейдите в SQL Editor
3. Скопируйте и выполните содержимое файла `supabase/delete_glutes_category.sql`
4. Проверьте результат - должно остаться только 7 категорий

## Проверка:

После удаления:
1. Обновите страницу приложения (F5)
2. Откройте дневник тренировок
3. Нажмите кнопку "+" (Добавить упражнение)
4. Убедитесь, что категории "ЯГОДИЦЫ" нет в списке
5. Должны быть только: ПЛЕЧИ, РУКИ, ГРУДЬ, СПИНА, НОГИ, ПРЕСС, КАРДИО
