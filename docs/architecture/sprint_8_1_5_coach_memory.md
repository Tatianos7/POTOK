# Sprint 8.1.5 — AI Coach Long‑Term Memory

Цель: сформировать архитектуру долговременной памяти и эволюции отношений между
пользователем и AI‑коучем, чтобы коуч помнил историю, учитывал прогресс и
мягко адаптировал тон поддержки.

## Memory Layers
- **Short‑term** (день/неделя): контекст текущего состояния и событий.
- **Mid‑term** (фазы/привычки/ритм): устойчивые паттерны недели/месяца.
- **Long‑term** (идентичность/характер/история пути): цели, ценности, долгий путь.

## Типы памяти
- **Событийная:** дни, тренировки, питание, замеры.
- **Эмоциональная:** стресс, уверенность, усталость, мотивация.
- **Доверия:** adherence, отклики, откаты, recovery.
- **Отношений:** тон, границы, стиль поддержки.

## Relationship Model
**Стадии отношений**
- onboarding
- trust building
- stable partnership
- relapse & recovery
- long‑term companion

**Связь с Calm Power Coach**
- Тон меняется от мягкой поддержки к уверенной силе.
- Глубина сообщений растет по мере доверия.
- Частота вмешательств уменьшается при росте автономии.

## Интеграция с UI
Связь памяти с экранами:
- **Today**: контекст дня и мягкие подсказки.
- **Progress**: история пути и мотивационные циклы.
- **Habits**: паттерны ритма, срывы и восстановление.
- **Relapse / Recovery**: восстановительный сценарий без давления.
- **Coach UI Components** из Sprint 8.1.4.

## Definition of Done
- Описана архитектура памяти и отношений.
- Зафиксированы слои памяти и типы сигналов.
- Связано с Calm Power Coach UX.
- Готово к переходу в реализацию Sprint 8.2.
