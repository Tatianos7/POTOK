# Команды для сохранения изменений в Git и деплоя на GitHub Pages

## 1. Сохранение в Git

Выполни следующие команды в терминале:

```bash
cd /Users/urijurij/Desktop/POTOK

# Добавить все изменения
git add -A

# Создать коммит
git commit -m "feat: улучшена работа дневника питания - оптимистичные обновления, встроенный календарь, исправлены ошибки Supabase"

# Отправить в GitHub
git push
```

## 2. Деплой на GitHub Pages

Если у тебя настроен GitHub Actions (файл .github/workflows/deploy.yml), то после push изменения автоматически задеплоятся.

Или выполни вручную:

```bash
# Собрать проект
npm run build

# Если установлен gh-pages
npx gh-pages -d dist
```
