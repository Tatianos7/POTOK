# Обновление категории "Ноги" - удаление категории "Ягодицы"

1. ✅ **Удалена категория "Ягодицы"** из SQL схемы и seed скрипта
2. ✅ **Все упражнения перенесены в категорию "Ноги"** с правильными целевыми мышцами
3. ✅ **Обновлены все упражнения на ноги** согласно предоставленному списку
4. ✅ **Обновлен UI компонент** для отображения 7 категорий вместо 8

## Мышцы для фильтра категории "Ноги":

В фильтре по мышцам для категории "Ноги" теперь доступны:
- **Большая ягодичная**
- **Средняя ягодичная**
- **Малая ягодичная**
- **Ягодичная** (общая)
- **Ягодицы** (общая)
- **Икроножные мышцы**
- **Квадрицепс**
- **Бицепс бедра**
- **Приводящие** / **Приводящие мышцы**
- **Поясница** (для становой тяги)

## Структура категорий:

- **Первая строка:** ПЛЕЧИ, РУКИ, ГРУДЬ
- **Вторая строка:** СПИНА, НОГИ, ПРЕСС
- **Третья строка:** КАРДИО (1 колонка), МОИ УПРАЖНЕНИЯ (2 колонки)

## Применение изменений:

### Способ 1: Автоматическое (рекомендуется)

1. Запустите импорт упражнений через страницу `/import-exercises`
2. Категория "Ягодицы" будет автоматически удалена, если она существует
3. Все упражнения будут импортированы с правильными целевыми мышцами

### Способ 2: Вручную через SQL Editor

Если категория "Ягодицы" уже существует в базе данных, выполните скрипт `supabase/remove_glutes_category.sql` в Supabase SQL Editor перед импортом упражнений.

## Проверка:

После применения изменений:
1. Откройте страницу "Дневник тренировок"
2. Нажмите кнопку "+" (Добавить упражнение)
3. Выберите категорию "НОГИ"
4. Проверьте, что все упражнения отображаются с правильными целевыми мышцами
5. Нажмите на иконку фильтра рядом с поиском
6. Убедитесь, что в фильтре доступны все мышцы, включая:
   - Большая ягодичная
   - Средняя ягодичная
   - Малая ягодичная
   - Ягодичная
   - Икроножные мышцы
