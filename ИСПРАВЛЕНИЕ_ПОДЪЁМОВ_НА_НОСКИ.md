# Исправление целевых мышц для упражнений на подъёмы на носки

## Проблема:
Упражнения "Подъёмы на носки" не отображали целевую мышцу "Икроножные мышцы" в списке упражнений и фильтре.

## Исправлено:
1. ✅ Обновлен seed скрипт для правильного обновления связей упражнений с мышцами
2. ✅ Все три упражнения правильно указаны в seed данных с мышцей "Икроножные мышцы"
3. ✅ Мышца "Икроножные мышцы" добавлена в SQL схему

## Упражнения:
- **Подъёмы на носки сидя или стоя** → Икроножные мышцы
- **Подъёмы на носки у стены** → Икроножные мышцы  
- **Подъёмы на носки с гантелями** → Икроножные мышцы

## Решение:

### Способ 1: Автоматическое (рекомендуется)

1. Запустите импорт упражнений через страницу `/import-exercises`
2. Seed скрипт автоматически:
   - Найдет существующие упражнения
   - Удалит старые связи с мышцами
   - Создаст правильные связи с мышцей "Икроножные мышцы"
3. После импорта упражнения будут отображаться с правильной целевой мышцей

### Способ 2: Вручную через SQL Editor

Если импорт не помог, выполните SQL скрипт `supabase/fix_calf_exercises.sql` в Supabase SQL Editor. Этот скрипт:
- Проверит наличие мышцы "Икроножные мышцы"
- Найдет все три упражнения на подъёмы на носки
- Обновит их связи с правильной мышцей

## Проверка:

После применения изменений:
1. Откройте страницу "Дневник тренировок"
2. Нажмите кнопку "+" (Добавить упражнение)
3. Выберите категорию "НОГИ"
4. Найдите упражнения:
   - "Подъёмы на носки сидя или стоя"
   - "Подъёмы на носки у стены"
   - "Подъёмы на носки с гантелями"
5. Убедитесь, что каждое упражнение отображается в формате: **"Название — Икроножные мышцы"**
6. Нажмите на иконку фильтра рядом с поиском
7. Убедитесь, что в фильтре доступна мышца **"Икроножные мышцы"**
8. При фильтрации по "Икроножные мышцы" должны отображаться все три упражнения

## Изменения в коде:

### `src/utils/seedExercises.ts`:
- Теперь скрипт удаляет все старые связи упражнения с мышцами перед созданием новых
- Это гарантирует, что связи будут актуальными даже для существующих упражнений
- Добавлено логирование для отслеживания создания связей

### `src/data/exercisesSeed.ts`:
- Все три упражнения правильно указаны с мышцей "Икроножные мышцы"

### `supabase/workout_schema.sql`:
- Мышца "Икроножные мышцы" добавлена в список мышц
